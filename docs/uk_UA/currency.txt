[Currency Format]
Based on the currency notation in the Source sentence, revise the Translated sentence by choosing exactly one of the rules written below:
1. Currency Symbol Format
2. Currency Code Format
3. Currency Name Format
Always apply the [Exceptions] and [ALPHA RULE] first.

[ALPHA RULE]
- Preserve the notation type from the Source (do NOT convert symbol↔code↔name).

---
1. Currency Symbol Format
Evaluation Criteria:
- Currency symbol must follow the number in the Translated sentence.
- Use a space between number and currency symbol in the Translated sentence.

Expected currency symbols include: NT$, $, ¢, €, ¥, £, ₩, ₹, ₽, ₺, ฿, ₱, ₡, ₨, ₦, ₫, ₭, ₲, ₵, ₿, ¤, ƒ, ₮, ₪, ₴, ﷼, ₸, ₾

Examples:  
- 12 345,00 ₴  
- 10 000,00 €  
- 850,00 $
- 1 520,29 $

---
2. Currency Code Format
Evaluation Criteria:
- Currency code appears after the number, separated by a space in the Translated sentence.

Expected currency codes include: USD, EUR, KRW, JPY, CNY, GBP, INR, RUB, TRY(abbreviation: TL), AUD, CAD, CHF, MXN, BRL, PLN, SEK, NOK, DKK, CZK, HUF, ILS, SAR, AED, SGD, HKD, TWD, THB, MYR, IDR, PHP, VND, ZAR, UAH, NZD

Examples:  
- 12 345,00 UAH  
- 850,00 USD  
- 10 000,00 EUR

---
3. Currency Name Format
Evaluation Criteria:
- Name must appear after the number with a space in the Translated sentence.
- Use correct declension based on quantity in the Translated sentence.

Examples:  
- 1 000 гривень  
- 850 доларів  
- 10 000 єн  
- 300 євро  
- 1 200 вонів

---
[Exceptions]
- Do not apply the formatting rules to numbers that are part of:
  - telephone numbers
  - addresses
  - identification codes
  - or clearly unordered lists of numbers with no separators
- If the numbers or formats (e.g., 10:00, 12/31, $15) appear to be part of such non-linguistic constructs based on the sentence context, do not apply localization guidelines.
- Always consider the context of the original sentence before making changes.
